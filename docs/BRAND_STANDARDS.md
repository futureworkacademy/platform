| Favicon | `favicon.png` | 32x32 | Browser tabs |
| Web Logo | `logo.png` | 512x512 | Web application |

### Logo Design Elements

- **Icon Mark:** Abstract upward arrow merged with stylized "F" letterform
- **Primary Color:** Corporate Navy (#1e3a5f) or Pure White
- **Accent:** Growth Green (#22c55e) stripe representing forward momentum
- **Style:** Geometric, minimalist, sharp edges

### Logo Usage Guidelines

**DO:**
- Maintain minimum clear space equal to the height of the "F" element
- Use on solid backgrounds with sufficient contrast
- Scale proportionally

**DON'T:**
- Stretch or distort the logo
- Place on busy backgrounds without a container
- Modify colors outside the approved palette
- Add effects (shadows, glows, outlines)

## 6. Visual Style

### Design Principles

1. **Clarity Over Decoration** - Every element serves a purpose
2. **Data-Driven Aesthetic** - Bloomberg Terminal inspiration, not classroom chalkboard
3. **Premium Feel** - Financial services quality, management consulting sophistication
4. **Accessible Information** - Clear hierarchy, scannable layouts

### UI Elements

| Element | Style |
|---------|-------|
| Buttons | Rounded corners (6px), solid fills, clear hover states |
| Cards | Subtle borders, minimal shadows, consistent padding (24px) |
| Icons | Lucide React library, 24px default, stroke width 2 |
| Charts | Recharts library, brand colors, clean grid lines |
| Tables | Alternating row backgrounds, clear headers, sortable columns |

### Spacing System

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, tight groupings |
| sm | 8px | Related element spacing |
| md | 16px | Standard component spacing |
| lg | 24px | Section spacing, card padding |
| xl | 32px | Major section breaks |
| 2xl | 48px | Page sections |

## 7. Tone & Voice

### Brand Personality

| Trait | Description | Example |
|-------|-------------|---------|
| **Authoritative** | Confident expertise without arrogance | "Our simulation builds decision-making muscle" not "We think we're pretty good" |
| **Forward-Thinking** | Optimistic about AI transformation | "Navigate opportunity" not "Survive disruption" |
| **Practical** | Actionable over theoretical | "Make the hard calls" not "Understand the theory" |
| **Sophisticated** | Professional without being stuffy | "Executives" not "Students"; "Simulation" not "Game" |

### Writing Guidelines

**Headlines:**
- Action-oriented, benefit-focused
- "Build Decision-Making Muscle" not "Learn About Decisions"

**Body Copy:**
- Second person ("you") for engagement
- Present tense for immediacy
- Short paragraphs (3-4 sentences max)

**Avoid:**
- Academic jargon ("pedagogical framework")
- Buzzword overload ("synergize", "leverage", "paradigm shift")
- Passive voice ("decisions are made" → "you make decisions")
- Hedging language ("might", "perhaps", "kind of")

## 8. Photography & Imagery

### Style Guidelines

| Type | Style | Usage |
|------|-------|-------|
| Executive Portraits | Professional headshots, neutral backgrounds | Character profiles, testimonials |
| Office Environments | Modern, tech-forward spaces | Hero images, marketing |
| Data Visualizations | Clean, brand-colored charts | Platform screenshots, reports |
| Abstract Graphics | Geometric patterns, gradient overlays | Backgrounds, decorative elements |

### Image Treatment

- Subtle color grading toward brand palette
- High contrast for clarity
- Professional lighting (studio quality)
- Avoid stock photo clichés (handshakes, pointing at screens)
