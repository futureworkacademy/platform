## Part 3: Where Content Goes (Step-by-Step)

### Adding Text Content (Briefings, Reports, Intel)

1. **Navigate to:** Admin Dashboard → Simulation Content
   - URL: `/admin/simulation-content`

2. **Select the module:** "AI Workplace Transformation"

3. **Choose content type:**
   - `Briefing` for weekly scenarios
   - `Research` for pre-game reports
   - `Intel` for news articles

4. **Fill in fields:**
   - **Title:** The headline
   - **Content:** Paste your text (Markdown supported)
   - **Week Number:** Which week it appears (0 for pre-game research)
   - **Category:** Optional grouping
   - **Order:** Display sequence within the week

5. **Save and verify** by previewing student view

---

### Adding Character Profiles

1. **Navigate to:** Admin Dashboard → Character Profiles
   - URL: `/admin/character-profiles`

2. **Create new character:**
   - **Name:** Character's full name
   - **Role:** Their position (e.g., "CFO", "Union Organizer")
   - **Bio:** 150-200 word background
   - **Stance:** Their general attitude
   - **Traits:** Influence (1-10), Hostility (1-10), Flexibility (1-10), Risk Tolerance (1-10)

3. **Generate or upload headshot:**
   - Use the built-in AI headshot generator, OR
   - Upload your own image (512x512 recommended)

---

### Adding Media Files (Images, Audio, Video)

1. **Upload to Object Storage:**
   - Navigate to the Object Storage pane in Replit
   - Upload your file to the `public` folder
   - Copy the URL

2. **Link in Content Editor:**
   - For videos: Paste URL in `Embed URL` or `Media URL` field
   - For audio: Paste URL in `Media URL` field
   - For images: Paste URL in `Thumbnail URL` field

3. **For character headshots:**
   - Use the built-in generator (recommended), OR
   - Upload image, copy URL, paste in profile

## Part 4: AI Prompt Templates

Use these prompts with Gemini, Claude, or ChatGPT to generate content based on GAME_DESIGN.md.

### Prompt: Weekly Briefing Generator

```
You are writing immersive business simulation content for graduate MBA students.

CONTEXT:
- Company: Apex Manufacturing, mid-sized automotive parts supplier
- 2,400 employees, $125M revenue
- Facing pressure to adopt AI/automation
- Core tension: financial performance vs. workforce wellbeing

WEEK [X]: [TITLE]

Based on this scenario outline:
[PASTE THE WEEK'S SECTION FROM GAME_DESIGN.md]

Write a 600-800 word briefing narrative that:
1. Opens with an urgent situation or stakeholder pressure
2. Describes the current state of the company this week
3. Introduces 2-3 conflicting stakeholder perspectives
4. Presents the key question/decision students must address
5. Uses professional but engaging business writing
6. Includes specific numbers, quotes from stakeholders, and concrete details

Tone: Serious, realistic, slightly tense. Like a Harvard Business School case study meets a Netflix drama.
```

---

### Prompt: Research Report Generator

```
You are writing educational research content for a business simulation platform.

Write a comprehensive research report titled: "[TITLE]"

Based on this outline:
[PASTE THE REPORT'S SECTION FROM GAME_DESIGN.md]

Requirements:
- 1,200-1,500 words
- Include an executive summary (3-4 sentences)
- Use headers and subheaders for organization
- Include 5-8 specific statistics with realistic sources (can be fictional but plausible)
- Include at least one table or data visualization description
- Write for MBA-level readers
- End with "Key Takeaways" bullet points

Tone: Authoritative, data-driven, like a McKinsey or Harvard Business Review article.
```

```

- Include 2-3 quotes from fictional experts or executives
- Reference recent trends or studies (can be fictional but realistic)

Tone: Professional journalism, like Wall Street Journal or Bloomberg.
```

## Part 5: Quality Checklist

Before marking any content as complete, verify:

### For Briefings
- [ ] 500-800 words
- [ ] Opens with urgency/tension
- [ ] Includes specific stakeholder quotes
- [ ] Contains concrete numbers/metrics
- [ ] Ends with clear decision question
- [ ] No placeholder text like "[INSERT X]"
- [ ] Proofread for typos

### For Research Reports
- [ ] 1,200-1,800 words
- [ ] Has executive summary
- [ ] Includes 5+ statistics
- [ ] Organized with headers
- [ ] Ends with key takeaways
- [ ] Professional tone throughout

### For Intel Articles
- [ ] 400-600 words
- [ ] Has compelling headline
- [ ] Includes expert quotes
- [ ] Relevant to that week's theme
- [ ] Standalone (doesn't require other context)

### For Character Profiles
- [ ] 150-200 word bio
- [ ] Includes career background
- [ ] Explains their perspective on AI transition
- [ ] Traits set (influence, hostility, flexibility, risk tolerance)
- [ ] Headshot uploaded or generated
- [ ] Quote style defined

## Part 6: Content Dependencies

Some content references other content. Create in this order:

```
1. CHARACTER PROFILES (first)
   └── Characters are referenced in briefings and intel

2. RESEARCH REPORTS (second)
   └── Students read these before simulation starts

3. WEEKLY BRIEFINGS (third)
   └── Reference characters and can reference research

4. INTEL ARTICLES (fourth)
   └── Supplement each week's briefing

5. MEDIA (last)
   └── Videos, audio, images enhance existing text
```

## Part 8: Troubleshooting

### Content Not Appearing
1. Check `is_active` is set to true
2. Verify week number is correct (0 for pre-game)
3. Confirm module_id matches the simulation
4. Clear browser cache and refresh

### Media Not Loading
1. Verify file uploaded to `public` folder in Object Storage
2. Check URL is complete and accessible
3. Ensure file format is supported (MP4, MP3, PNG, JPG)
4. Check file size limits

### Character Headshot Not Generating
1. Ensure headshot prompt is detailed enough
2. Try adjusting prompt (remove brand names, specific features)
3. Alternative: Upload your own image
