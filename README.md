# Future Work Academy

  AI Business Simulation Platform — An educational simulation where graduate students navigate AI transformation at Apex Manufacturing over 8 weeks.

  Built with React, TypeScript, Express.js, and PostgreSQL.

  ## Features

  - 8-week simulation with branching scenarios and stakeholder dynamics
  - 17 AI-generated character profiles with quantifiable traits
  - LLM-powered essay grading with structured rubric criteria
  - Phone-a-Friend advisor system with AI-generated guidance
  - 3-tier difficulty framework (Introductory/Standard/Advanced)
  - Dual scoring: financial performance + cultural health
  - Real-time analytics and leaderboard
  - Self-service demo system with guided tour
  - Academic white paper and comprehensive methodology documentation
  - SEO infrastructure with GA4, sitemap, JSON-LD structured data

  ## Tech Stack

  - **Frontend:** React, TypeScript, Tailwind CSS, Shadcn/UI
  - **Backend:** Express.js, PostgreSQL, Drizzle ORM
  - **AI/ML:** OpenAI GPT-4o-mini, Google Gemini
  - **Auth:** Replit OIDC
  - **Notifications:** Twilio SMS, SendGrid Email
